# Grounder

A lightweight PostgreSQL client for developers who prefer simplicity

[![Download](https://img.shields.io/github/v/release/deshario/Grounder?color=22c55e&label=Download)](https://github.com/deshario/Grounder/releases/latest)

![Screenshot](https://i.gyazo.com/d5bd6c6755357e29b7898b5eee7ae561.png)

## Features

- **Query Runner** - Multi-block SQL execution with syntax highlighting
- **Table Management** - View, edit, create, and delete tables and rows
- **Filters** - Filter table data with conditions
- **Secure Storage** - Credentials stored in macOS Keychain

## Install

Download from [Releases](https://github.com/deshario/Grounder/releases/latest), then run:

```bash
xattr -cr /Applications/Grounder.app
```

## Development

```bash
pnpm install
pnpm dev
```

## Build

```bash
pnpm dist
```
